# Student Task & Deadline Tracker API

## Description
This is a backend REST API built with Django and Django REST Framework.
The system allows students to manage academic tasks and deadlines efficiently.

## Features
- User registration and authentication
- Create, update, and delete tasks
- Track deadlines
- Filter overdue tasks

## Tech Stack
- Python
- Django
- Django REST Framework

## Setup Instructions
1. Clone the repository
2. Create a virtual environment
3. Install requirements
4. Run migrations
5. Start the development server
